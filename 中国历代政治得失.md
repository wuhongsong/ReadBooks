

### 中国古代中央官制演变表


| 朝代 | 核心制度 | 架构特点简述 | 皇权 vs 相权（权力博弈） |
| :--- | :--- | :--- | :--- |
| **秦 · 汉初** | **三公九卿制** | 丞相统领百官，拥有独立决策权。 | **相权极重，制衡皇权。**<br>此时丞相有“坐而论道”的地位，是一人之下、万人之上。 |
| **汉** (武帝起) | **中外朝制** | **内朝**（皇帝亲信）决策，**外朝**（丞相九卿）执行。 | **相权受挫，皇权扩张。**<br>皇帝用“身边人”架空了“外面的丞相”，相权开始沦为执行者。 |
| **魏晋南北朝** | **三省雏形** | 尚书、中书、门下三省逐渐成型。 | **相权分散，过渡时期。**<br>权力从一个丞相向多个机构转移，为隋唐铺路。 |
| **隋 · 唐** | **三省六部制** | **中书**（定策）→**门下**（审核）→**尚书**（执行）。 | **相权三分，皇权独尊。**<br>把一个宰相的权力劈成三份，让他们互相吵架，皇帝居中裁决。 |
| **宋** | **二府三司制** | 行政、军事、财政三权分离。 | **相权肢解，皇权集权。**<br>不仅分了行政权，连军权（枢密院）和财权（三司）都被拿走了，宰相彻底文职化。 |
| **元** | **一省制** | 只设中书省，统领六部。 | **相权反弹，威胁皇权。**<br>为了效率恢复了“独相”，结果导致后期权相（如燕帖木儿）频出，皇权反而不稳。 |
| **明** | **废相·内阁** | **废除丞相**，六部直属皇帝。 | **相权灭亡，皇权极权。**<br>彻底消灭了“丞相”这个职位。内阁只是秘书，没有法定权力。 |
| **清** | **军机处** | 军机大臣跪受笔录。 | **皇权顶峰，不仅无相，也无“人”。**<br>大臣完全沦为皇帝的工具人，没有任何机构能制衡皇帝。 |


---
💡 为什么加上“汉武帝”这一行这么重要？

加上这一行后，你就能看清一条非常有趣的**历史规律**：

1.  **皇帝不信任“外人”：** 秦汉初期，丞相（外朝）权力太大。
2.  **皇帝用“身边人”夺权：** 汉武帝启用身边的秘书（尚书/侍中）组成**内朝**，架空了丞相。
3.  **“身边人”变成“外人”：** 到了魏晋隋唐，当年的秘书机构（尚书/中书/门下）慢慢变成了正式的政府机构（三省），变成了新的“外朝”。
4.  **皇帝再找新的“身边人”：** 明清时期，皇帝又觉得三省六部太正式了，于是又弄出了“内阁”和“军机处”这些新的秘书班子。


**结论：** 中国古代官制史，其实就是皇帝不断把**“秘书”**变成**“宰相”**，然后再找**“新秘书”**的过程。
**总结这条演变规律：**
皇权不断加强，相权不断削弱： 从秦朝丞相大权在握，到隋唐一分为三，到宋朝被分割军财权，到明朝直接废除，最后清朝皇帝独裁。
“六部”非常稳定： 虽然顶层的宰相制度一直在变（三公、三省、内阁），但干活的六部（吏户礼兵刑工）从隋唐确立后，一直延续到清末，几乎没有大变。



### 中国古代中央官制与权力博弈表（含宦官势力）

| 朝代 | 核心制度 | 皇权 vs 相权 (明线) | **宦官势力 (暗线)** | **权力格局一句话总结** |
| :--- | :--- | :--- | :--- | :--- |
| **秦 · 汉初** | **三公九卿** | **相权极重。** 丞相是真正的政府首脑。 | **萌芽期。** 赵高虽乱政，但属于个案，制度上宦官无权。 | **君相共治**<br>（皇帝与丞相一起坐着商量事） |
| **汉** (中后期) | **中外朝** | **皇权架空相权。** 汉武帝用内朝打压外朝。 | **寄生期。** 皇帝为了对抗外朝，开始用宦官传话（中书谒者）。**东汉末年宦官彻底掌权**，导致党锢之祸。 | **内廷外朝死斗**<br>（皇帝用太监斗官僚，结果太监骑到了头上） |
| **隋 · 唐** | **三省六部** | **相权三分。** 制度设计最完美的平衡。 | **爆发期。** 唐太宗时期严控，但**唐中后期宦官掌军权（神策军）**，甚至能废立皇帝（如杀唐宪宗）。 | **前期制度平衡，后期家奴弑主**<br>（三省制再好，挡不住手里有兵的太监） |
| **宋** | **二府三司** | **相权肢解。** 宰相彻底失去军权和财权。 | **受控期。** 宋朝制度设计严密，且重文轻武，宦官虽然领兵（如童贯），但**始终不敢废立君主**，被文官集团压制。 | **皇权独大，文官治国**<br>（太监有权，但翻不了天；宰相听话，但地位尊崇） |
| **明** | **废相·内阁** | **相权灭亡。** 废丞相，皇帝直接对接六部。 | **巅峰期（代理人）。** 皇帝懒得管六部，就让太监（司礼监）去管内阁（批红权）。**魏忠贤**等权阉出现，名为“九千岁”。 | **皇权不够，太监来凑**<br>（皇帝废了宰相，结果不得不依赖太监去控制文官） |
| **清** | **军机处** | **无相亦无人。** 军机大臣只是秘书。 | **消亡期。** 吸取明朝教训，立下铁牌“太监不得干政”。清朝**彻底杜绝了宦官干政**（除晚清慈禧时期安德海、李莲英稍有势头外）。 | **绝对皇权**<br>（主子就是主子，奴才就是奴才，没有中间商） |

### 深度解析：为什么会形成这个局面？

1.  **汉朝的教训（引狼入室）：**
    皇帝不相信丞相（外人），所以用身边的宦官（家奴）去传话、去夺权。结果家奴离皇帝太近，最后把皇帝架空了。

2.  **唐朝的悲剧（掌握军权）：**
    唐朝最大的漏洞是让宦官掌握了禁军（神策军）。一旦太监手里有了枪杆子，连皇帝都得看他们脸色。

3.  **明朝的逻辑（畸形平衡）：**
    朱元璋废了宰相，觉得痛快了。但他子孙没那么勤快，干不动活，又怕文官集团（内阁）骗自己，只好把自己那份“批改作业”的权力（批红）交给太监（司礼监）。
    *   **明朝的政治实质是：** 皇帝躲在幕后，指挥**“司礼监（太监）”**去打压**“内阁（文官）”**。太监越坏，皇帝往往越安全（因为太监除了依附皇权，没有退路）。

4.  **清朝的终结：**
    清朝皇帝最勤政，也最独裁。他们不需要宰相帮手，也不需要太监制衡文官。因为在满清制度下，所有官员都是“奴才”，皇帝一人乾纲独断。

这个表格把**明面上的官制**和**暗地里的太监政治**结合起来，应该就是中国古代政治生态的最完整写照了。
