

### 中国古代中央官制演变表


| 朝代 | 核心制度 | 架构特点简述 | 皇权 vs 相权（权力博弈） |
| :--- | :--- | :--- | :--- |
| **秦 · 汉初** | **三公九卿制** | 丞相统领百官，拥有独立决策权。 | **相权极重，制衡皇权。**<br>此时丞相有“坐而论道”的地位，是一人之下、万人之上。 |
| **汉** (武帝起) | **中外朝制** | **内朝**（皇帝亲信）决策，**外朝**（丞相九卿）执行。 | **相权受挫，皇权扩张。**<br>皇帝用“身边人”架空了“外面的丞相”，相权开始沦为执行者。 |
| **魏晋南北朝** | **三省雏形** | 尚书、中书、门下三省逐渐成型。 | **相权分散，过渡时期。**<br>权力从一个丞相向多个机构转移，为隋唐铺路。 |
| **隋 · 唐** | **三省六部制** | **中书**（定策）→**门下**（审核）→**尚书**（执行）。 | **相权三分，皇权独尊。**<br>把一个宰相的权力劈成三份，让他们互相吵架，皇帝居中裁决。 |
| **宋** | **二府三司制** | 行政、军事、财政三权分离。 | **相权肢解，皇权集权。**<br>不仅分了行政权，连军权（枢密院）和财权（三司）都被拿走了，宰相彻底文职化。 |
| **元** | **一省制** | 只设中书省，统领六部。 | **相权反弹，威胁皇权。**<br>为了效率恢复了“独相”，结果导致后期权相（如燕帖木儿）频出，皇权反而不稳。 |
| **明** | **废相·内阁** | **废除丞相**，六部直属皇帝。 | **相权灭亡，皇权极权。**<br>彻底消灭了“丞相”这个职位。内阁只是秘书，没有法定权力。 |
| **清** | **军机处** | 军机大臣跪受笔录。 | **皇权顶峰，不仅无相，也无“人”。**<br>大臣完全沦为皇帝的工具人，没有任何机构能制衡皇帝。 |


---
💡 为什么加上“汉武帝”这一行这么重要？

加上这一行后，你就能看清一条非常有趣的**历史规律**：

1.  **皇帝不信任“外人”：** 秦汉初期，丞相（外朝）权力太大。
2.  **皇帝用“身边人”夺权：** 汉武帝启用身边的秘书（尚书/侍中）组成**内朝**，架空了丞相。
3.  **“身边人”变成“外人”：** 到了魏晋隋唐，当年的秘书机构（尚书/中书/门下）慢慢变成了正式的政府机构（三省），变成了新的“外朝”。
4.  **皇帝再找新的“身边人”：** 明清时期，皇帝又觉得三省六部太正式了，于是又弄出了“内阁”和“军机处”这些新的秘书班子。


**结论：** 中国古代官制史，其实就是皇帝不断把**“秘书”**变成**“宰相”**，然后再找**“新秘书”**的过程。
**总结这条演变规律：**
皇权不断加强，相权不断削弱： 从秦朝丞相大权在握，到隋唐一分为三，到宋朝被分割军财权，到明朝直接废除，最后清朝皇帝独裁。
“六部”非常稳定： 虽然顶层的宰相制度一直在变（三公、三省、内阁），但干活的六部（吏户礼兵刑工）从隋唐确立后，一直延续到清末，几乎没有大变。
