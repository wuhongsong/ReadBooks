## 开篇：技术分析到底是什么？

### 技术分析的三个基本假设

技术分析建立在三个假设之上（注意：是"假设"，不是"真理"）：

| 假设 | 含义 | 你该怎么理解 |
|---|---|---|
| **市场行为包容一切** | 所有影响股价的因素（业绩、政策、情绪、战争…）最终都会反映在价格和成交量上 | 不用去追消息，看图就够了（这是理想情况） |
| **价格沿趋势运动** | 上涨的趋势更可能继续上涨，下跌的趋势更可能继续下跌 | "顺势而为"，不要逆势抄底 |
| **历史会重演** | 人性不变，所以价格的运行模式会重复出现 | 过去有用的形态，未来可能还有用 |

### 技术分析能做什么，不能做什么

```
能做的：
  ✅ 告诉你市场现在"冷"还是"热"（情绪）
  ✅ 告诉你资金是在流入还是流出
  ✅ 给你一个参考的买卖区域（不是精确时点）
  ✅ 帮你管理风险（设置止损位）

不能做的：
  ❌ 精确预测明天涨跌
  ❌ 告诉你"这只股票值不值得买"（那是基本面分析的活）
  ❌ 在政策突变、黑天鹅事件时给出正确信号
```

### 本文的知识体系

```
技术分析
├── 趋势指标（告诉你：趋势是向上、向下、还是横盘？）
│   ├── 均线 MA / EMA
│   ├── MACD
│   └── 布林带 BOLL
│
├── 动量指标（告诉你：涨/跌的力气用完了没有？有没有超买/超卖？）
│   ├── RSI
│   ├── KDJ
│   ├── CCI
│   └── WR（威廉指标）
│
├── 成交量/资金指标（告诉你：有没有真金白银在进场？）
│   ├── 成交量 + 换手率
│   ├── OBV（能量潮）
│   └── 融资余额
│
└── 综合研判（把以上指标组合起来，形成判断）
```

## 模块一：趋势指标
> **核心问题**：股价现在的趋势是什么？是上涨、下跌，还是横着不走？

趋势指标是"滞后指标"——它们基于过去的价格计算，告诉你已经发生了什么趋势，而不是直接预测未来。

---

### 1.1 均线（MA —— Moving Average）

#### 📖 来龙去脉

**发明者**：没有特定发明者，是最古老的技术指标之一。
**发明逻辑**：股价的日常波动充满噪音，就像大海里的波浪。均线就是把过去N天的价格平均起来，画出一条"海平面"，让你看清楚潮汐的方向（大趋势），而不是被每一朵浪花干扰。

#### 🔢 怎么算的

```
MA_5  = （今天收盘 + 昨天收盘 + 前天收盘 + 大前天 + 大大前天）÷ 5
MA_20 = 过去20天收盘价的平均值
MA_60 = 过去60天收盘价的平均值（约等于一个季度的趋势）
```

#### 📊 福耀玻璃真实数据（2026-06-18）

| 指标 | 数值 | 含义 |
|---|---|---|
| 收盘价 | **49.95元** | — |
| MA_5（5日均线） | 51.21元 | 股价在下方 → 短期趋势向下 |
| MA_10（10日均线） | 52.17元 | 股价在下方 → 短期趋势向下 |
| MA_20（20日均线） | 52.79元 | 股价在下方 → 中期趋势向下 |
| MA_60（60日均线） | 55.55元 | 股价在下方 → 长期趋势向下 |
| MA_120（120日均线） | 57.76元 | 股价在下方 → 长期趋势向下 |
| MA_250（250日均线） | 59.52元 | 股价在下方 → 超长期趋势向下 |

**判断**：福耀玻璃当前处于**完全的"空头排列"**——所有均线都在股价上方，而且短期均线（MA5）在最下面，长期均线（MA250）在最上面。这是最典型的下跌趋势形态。

#### 💡 怎么看均线——三个核心信号

**信号1：金叉 vs 死叉**

```
金叉（买入信号）：
  MA_5 从下往上穿越 MA_20 → 短期趋势转强
  "短期平均成本开始超过中期平均成本，买盘在增加"

死叉（卖出信号）：
  MA_5 从上往下穿越 MA_20 → 短期趋势转弱
  "短期平均成本开始低于中期平均成本，卖盘在增加"

福耀玻璃当前状态：死叉已发生（MA5=51.21 < MA20=52.79）

**⚠️ 关键问题：金叉/死叉后会发生什么？（用福耀真实历史数据回答）**

很多教程只告诉你"金叉=买入，死叉=卖出"，但**没告诉你这两个信号的实际成功率**。我用福耀玻璃过去120天的真实数据，给你看客观结果：

| 信号类型 | 发生次数 | 成功次数 | 成功率 | 平均20天涨跌 |
|---|---|---|---|---|
| 金叉（MA5上穿MA20） | 3次 | 1次 | **33%** ❌ | -2.27% |
| 死叉（MA5下穿MA20） | 3次 | 3次 | **100%** ✅ | -4.25% |

**结论**：在福耀玻璃的下跌趋势中，**死叉信号极其可靠，金叉信号大多是假的**！

**为什么死叉比金叉更可靠？**

```
在下跌趋势中：
  死叉 = 趋势延续的信号 → 成功率高
  金叉 = 反弹信号，但可能不是趋势反转 → 成功率低（假信号多）

在上涨趋势中：
  金叉 = 趋势延续的信号 → 成功率高
  死叉 = 回调信号，但可能不是趋势反转 → 成功率低

福耀玻璃过去4个月 = 明确的下跌趋势
→ 所以死叉100%成功，金叉大多失败

实战含义：
  ✅ 在下跌趋势中：重视死叉，忽略金叉（金叉可能是陷阱）
  ✅ 在上涨趋势中：重视金叉，忽略死叉（死叉可能是回调）
  
  → 关键是：先判断"当前是上涨趋势还是下跌趋势"！
```

→ 死叉信号到目前为止是正确的

#### 💰 与资金流向的关系

- 当**资金持续流入**时，买盘推动股价站上均线 → 均线向上拐头
- 当**资金持续流出**时，卖盘压低股价跌破均线 → 均线向下拐头
- **福耀玻璃**：股价持续在所有均线下方，说明近期的资金净流出，卖方力量占优

#### ⚠️ 局限性

均线是**滞后指标**。等MA5下穿MA20形成"死叉"时，股价往往已经跌了一截了。所以均线更适合**确认趋势**，而不是**捕捉转折点**。

#### ⚠️ 初学者陷阱1：看到金叉就买，看到死叉就卖

```
错误做法：
  看到MA5上穿MA20形成金叉 → 立刻买入 ❌
  看到MA5下穿MA20形成死叉 → 立刻卖出 ❌

为什么错误？
  金叉可能是"假信号"（尤其是在下跌趋势中）
  死叉可能是"诱空"（尤其是在上涨趋势中）
  
正确做法：
  1. 先判断当前是上涨趋势还是下跌趋势
  2. 在上涨趋势中：重视金叉，谨慎对待死叉（可能是回调）
  3. 在下跌趋势中：重视死叉，谨慎对待金叉（可能是反弹）
  4. 用成交量验证：金叉+放量=可靠，金叉+缩量=可疑
  
福耀玻璃实战：
  2026年过去4个月：3次金叉，2次失败（成功率33%）
  → 在下跌趋势中，不要看到金叉就买！
```

#### ⚠️ 初学者陷阱2：只看均线，不看趋势

```
错误做法：
  只看MA5和MA20的关系，不看MA60和MA250 ❌
  
为什么错误？
  MA5和MA20是短期均线，只能看5-20天的趋势
  MA60和MA250是中长期均线，决定了大方向
  
正确做法：
  先看MA250（年线）= 牛熊分界
  再看MA60（季线）= 中期趋势
  最后看MA5和MA20 = 短期买卖点
  
福耀玻璃实战：
  MA250=59.52元，当前价格49.95元，远低于年线
  → 明确的长期下跌趋势，短期金叉都不可靠
```

#### 🔍 你问的关键问题：如果排列不是完美递增/递减怎么办？

你举的例子：`MA_5 < MA_10 > MA_20 < MA_60`

**这种情况在真实市场中经常出现！** 它叫做"均线纠缠"或"均线混乱排列"，往往发生在趋势转换的过渡期。

**福耀玻璃的真实案例**（基于真实数据计算）：

```
2026-06-02：收盘=54.24元
  MA5=52.82  MA10=53.53  MA20=55.36  MA60=56.56
  排列：价>MA5，但MA5<MA10<MA20<MA60
  → 股价短暂反弹到MA5上方，但中期均线仍在压制
  → 这是"反弹未遂"的信号，不是趋势反转

2026-06-05：收盘=53.60元
  MA5=53.54  MA10=53.24  MA20=54.66  MA60=56.36
  排列：价>MA5，MA5>MA10，但MA10<MA20<MA60
  → MA5刚上穿MA10（金叉），但MA10仍在MA20下方
  → 短期转强，但中期仍弱，典型的"熊市反弹"

2026-06-08：收盘=53.15元
  MA5=53.67  MA10=53.17  MA20=54.40  MA60=56.29
  排列：价<MA5，MA5>MA10，但MA10<MA20<MA60
  → 股价跌破MA5，反弹结束，重新进入下跌
```

**怎么解读混乱排列**：

| 排列类型 | 市场含义 | 操作建议 |
|---|---|---|
| 价>MA5，但MA5<MA10 | 短期反弹，但中期仍弱 | 不要追高，可能是诱多 |
| MA5>MA10，但MA10<MA20 | 短期转强，但中期压制 | 反弹高度有限，快进快出 |
| 价<MA5，且MA5<MA10<MA20 | 下跌趋势确认 | 不要抄底，等待反转信号 |

**关键结论**：**完美的多头/空头排列很少持续很久，混乱排列才是市场的常态**。你需要看的是"主要趋势"（MA60的方向），而不是被短期的混乱排列干扰。

### 1.2 EMA（指数移动平均线）

#### 📖 来龙去脉

**问题**：普通均线（MA）有一个缺陷——它对所有日子"一视同仁"。20天前的价格和昨天的价格，在MA计算里权重一样。

**解决**：EMA给最近的价格更高的权重，对更远的价格给更低的权重。这样EMA对最新价格的反应更快，滞后性比MA小。

#### 🔢 怎么算的（简化版）

```
EMA_12 = 昨天EMA_12 × (1 - 2/13) + 今天收盘价 × (2/13)
（给最近的价格约15.4%的权重，越远越小）
```

#### 📊 福耀玻璃真实数据

| 指标 | 数值 |
|---|---|
| EMA_12 | 52.04元 |
| EMA_26 | 53.34元 |
| EMA_50 | 54.76元 |

收盘价49.95元，同样在所有EMA下方 → 确认下跌趋势。

### 1.3 MACD（指数平滑异同移动平均线）

#### 📖 来龙去脉

**发明者**：Gerald Appel，1970年代。
**发明逻辑**：均线的"金叉死叉"信号太敏感，容易出现假信号。Appel想了一个办法：
1. 用两条不同速度的EMA（12日和26日）的差值（DIF）来捕捉趋势
2. 再对DIF做一条信号线（DEA），类似对DIF再做一次"均线"
3. 用DIF和DEA的关系来判断买卖信号
4. 用柱状图（MACD柱）来直观显示两者的差距

简单来说：**MACD是衡量"两条均线的距离"是否在扩大的指标**。

#### 🔢 怎么算的（简化版）

```
DIF = EMA_12 - EMA_26          ← 快线减慢线，反映短期趋势强弱
DEA = DIF的9日指数平均         ← 对DIF再做平滑，相当于"均线的均线"
MACD柱 = (DIF - DEA) × 2      ← 柱状图，正值在零轴上方，负值在下方
```

#### 📊 福耀玻璃真实数据（2026-06-18）

| 指标 | 数值 | 含义 |
|---|---|---|
| DIF | **-1.29** | 快线，负值说明短期均线在长期均线下方 |
| DEA | **-1.10** | 信号线，同样为负 |
| MACD柱 | **-0.39** | 负值，柱状图在零轴下方 |

**判断**：
- DIF和DEA都在零轴下方 → **空头市场**
- DIF < DEA → **死叉状态**，卖出信号
- MACD柱为负且走阔 → 下跌动能还在增强

#### 💡 怎么看MACD——三个核心信号

**信号1：零轴上方 vs 零轴下方**

```
零轴上方：多头市场，上涨趋势
零轴下方：空头市场，下跌趋势

福耀玻璃：DIF=-1.29，在零轴下方 → 明确的空头市场
```

**信号2：金叉 vs 死叉（在MACD里是指DIF和DEA的关系）**

```
MACD金叉：DIF从下往上穿越DEA → 买入信号
MACD死叉：DIF从上往下穿越DEA → 卖出信号

福耀玻璃：DIF(-1.29) < DEA(-1.10) → 死叉状态，且死叉已持续一段时间
```

**信号3：底背离（这是MACD最有价值的信号）**

```
底背离：股价创出新低，但MACD柱没有创出新低（在走高）
→ 说明下跌动能衰竭，可能即将反弹

顶背离：股价创出新高，但MACD柱没有创出新高（在走低）
→ 说明上涨动能衰竭，可能即将回调

福耀玻璃真实数据判断（基于120天历史计算）：
  最近两个股价低点：
    低点1（2026-04-30）：股价57.67元，DIF=+0.279
    低点2（2026-05-28）：股价51.99元，DIF=-1.254
  
  结论：❌ 不存在底背离！
  原因：股价创新低（57.67 → 51.99），DIF也创新低（+0.279 → -1.254）
        DIF和股价同步创新低 → 下跌趋势未结束，动能未衰竭
```

#### 💰 与资金流向的关系

- MACD金叉 → 短期资金流入速度加快，买盘增强
- MACD死叉 → 短期资金流出速度加快，卖盘增强
- **福耀玻璃**：死叉状态，说明资金仍在净流出

#### ⚠️ 重要提醒：MACD底背离需要至少两个低点才能判断

很多初学者看到一个低点就急着说"底背离"，这是错误的。**必须等股价创出第二个低点，才能和第一个低点比较**。

如果第二个低点还没出现，应该说"**有待观察**"，而不是"**存在底背离**"。

#### ⚠️ 初学者陷阱3：看到MACD金叉就买，不看趋势

```
错误做法：
  看到DIF上穿DEA形成金叉 → 立刻买入 ❌
  
为什么错误？
  在下跌趋势中，MACD金叉的失败率很高
  福耀玻璃案例：过去2次MACD金叉，0次成功（成功率0%）
  
正确做法：
  1. 先看DIF是否在零轴上方（零轴上方的金叉更可靠）
  2. 再看趋势：上涨趋势中的金叉 > 下跌趋势中的金叉
  3. 最后看成交量：金叉+放量 = 可靠，金叉+缩量 = 可疑
  
福耀玻璃实战：
  当前DIF=-1.29，在零轴下方
  → 即使出现金叉，也可能是下跌中的反弹，不是趋势反转
```

#### ⚠️ 初学者陷阱4：误把"金叉/死叉"当唯一买卖依据

```
错误做法：
  只盯着MACD金叉死叉，其他都不看 ❌
  
为什么错误？
  MACD金叉死叉只是"趋势确认"工具，不是"抄底逃顶"工具
  在震荡市中，MACD会频繁出现假信号
  
正确做法：
  MACD用来：确认趋势方向、判断动能强弱
  买卖决策需要：MACD + 均线 + 成交量 三者配合
  
实战原则：
  ✅ MACD金叉 + 均线多头排列 + 放量 = 强买入信号
  ✅ MACD死叉 + 均线空头排列 + 放量 = 强卖出信号
  ❌ 单独的MACD金叉/死叉 = 不可靠
```

#### ⚠️ 初学者陷阱5：忽视MACD柱的变化

```
很多初学者只盯着DIF和DEA的交叉，忽视了MACD柱！

MACD柱其实比DIF/DEA交叉更早给出信号：
  - MACD柱由负转正 → 下跌动能减弱（比金叉早出现）
  - MACD柱由正转负 → 上涨动能减弱（比死叉早出现）
  
福耀玻璃实战（2026-06-18）：
  MACD柱 = -0.39，为负且走阔
  → 说明下跌动能还在增强，还没有见底迹象
  → 不要急着抄底！
```

### 1.4 布林带（BOLL —— Bollinger Bands）

#### 📖 来龙去脉

**发明者**：John Bollinger，1980年代。
**发明逻辑**：均线是一条线，但它没有告诉你"波动有多大"。有的股票每天波动1%，有的每天波动5%，同样的价格信号，含义完全不同。

Bollinger想：**用一个"通道"来同时表示"趋势方向"和"波动大小"**。
- 中轨 = 均线（趋势方向）
- 上轨 = 中轨 + 2倍标准差（统计上的"正常波动上限"）
- 下轨 = 中轨 - 2倍标准差（统计上的"正常波动下限"）

**统计学含义**：在正态分布假设下，约95%的股价应该落在布林带通道内。如果股价跑出通道，说明"异常波动"，可能即将回归。

#### 🔢 怎么算的（简化版）

```
中轨 = MA_20（20日均线）
上轨 = 中轨 + 2 × 过去20日收盘价的标准差
下轨 = 中轨 - 2 × 过去20日收盘价的标准差

（标准差 = 衡量过去20天股价波动程度的统计量）
```

#### 📊 福耀玻璃真实数据（2026-06-18）

| 指标 | 数值 | 含义 |
|---|---|---|
| 收盘价 | **49.95元** | — |
| BOLL中轨 | 52.79元 | 20日均线 |
| BOLL上轨 | 55.28元 | 正常波动上限 |
| BOLL下轨 | **50.30元** | 正常波动下限 |

**关键判断**：
- 收盘价49.95元 **< 下轨50.30元**
- → 股价**跌破布林带下轨**
- → 统计意义上属于"异常波动"，短期有回归中轨的倾向（反弹需求）

#### 💡 怎么看布林带——四个核心信号

**信号1：股价与轨道的位置关系**

```
股价触及/突破上轨 → 短期超买，可能回调
股价触及/跌破下轨 → 短期超卖，可能反弹

福耀玻璃：跌破下轨 → 超卖信号（和RSI的信号一致！）
```

**信号2：布林带收口 vs 开口**

```
收口（通道变窄）：波动率降低，往往在积蓄力量，即将选择方向
开口（通道变宽）：波动率放大，趋势在加速

福耀玻璃：需要看过去一段时间的布林带宽度来判断
```

**信号3：中轨的方向**

```
中轨向上 → 中期趋势向上
中轨向下 → 中期趋势向下

福耀玻璃：中轨=MA20=52.79，且MA20 < MA60，中轨方向向下 → 中期趋势向下
```

**信号4：W底 / M顶（在布林带中的特殊形态）**

```
W底：股价两次触及下轨后反弹，形成"双底" → 买入信号
M顶：股价两次触及上轨后回落，形成"双顶" → 卖出信号
```

#### 💰 与资金流向的关系

- 股价触及下轨 + 成交量放大 → 有资金在低位接盘，反弹可信度高
- 股价触及下轨 + 成交量萎缩 → 可能是"阴跌"，反弹力度弱
- **福耀玻璃**：跌破下轨，需要看接下来几天是否出现"触及下轨后反弹 + 放量"的组合信号

#### ⚠️ 初学者陷阱6：看到股价突破布林带上轨就追涨

```
错误做法：
  股价突破上轨 → 认为会继续涨，追进去 ❌
  
为什么错误？
  突破上轨 = 超买信号，不是买入信号！
  统计上，95%的情况下股价会回到通道内
  
正确做法：
  突破上轨 → 警惕！可能即将回调
  等待股价回到通道内 + 其他指标确认 → 再决定买卖
  
福耀玻璃实战：
  过去3个月，股价一直在下轨附近，从未触及上轨
  → 当前是超卖，不是超买，不要搞反了！
```

#### ⚠️ 初学者陷阱7：看到股价跌破下轨就抄底

```
错误做法：
  股价跌破下轨 → 认为超卖，立刻抄底 ❌
  
为什么错误？
  跌破下轨确实说明超卖，但超卖可以继续超卖
  在强下跌趋势中，股价可以在下轨下方运行很久
  
正确做法：
  跌破下轨 → 观察
  等待：股价重新站上下轨 + 成交量放大 + RSI拐头向上
  → 三个条件同时满足，再考虑抄底
  
福耀玻璃实战（2026-06-18）：
  收盘价49.95 < 下轨50.30
  → 已经跌破下轨，但不要急着抄底
  → 等待反弹信号确认（RSI拐头、成交量放大）
```

#### ⚠️ 初学者陷阱8：忽视布林带的"收口"信号

```
很多初学者只盯着股价和轨道的关系，忽视了布林带的宽度变化！

布林带收口（通道变窄）：
  → 说明波动率降低，市场在选择方向的前夕
  → 接下来往往会有大行情（突破方向不确定）
  
布林带开口（通道变宽）：
  → 说明波动率放大，趋势在加速
  → 如果是上涨中开口 = 强势延续
  → 如果是下跌中开口 = 跌势加速
  
实战意义：
  ✅ 布林带极度收口 + 股价突破上轨 + 放量 = 强势上涨信号
  ❌ 布林带极度收口 + 股价突破下轨 + 放量 = 强势下跌信号
```

## 模块二：动量/超买超卖指标
> **核心问题**：这轮上涨/下跌的"力气"用完了没有？股价有没有涨太多（超买）或跌太多（超卖）？

动量指标是"震荡指标"——它们在固定范围内上下摆动，帮助你判断极端状态。

### 2.1 RSI（相对强弱指标 —— Relative Strength Index）

#### 📖 来龙去脉

**发明者**：J. Welles Wilder Jr.，1978年，著作《New Concepts in Technical Trading Systems》。
**发明逻辑**：Wilder发现，单纯看价格涨跌不够——有时候股价天天涨，但涨幅越来越小，说明"力气"用完了。他想要一个指标来量化"最近N天里，上涨的力量和下跌的力量，哪个更强？"

RSI的本质：**比较一段时间内上涨幅度和下跌幅度的平均值，用一个0~100的数字来表达"多空力量的对比"**。

#### 🔢 怎么算的（简化版）

```
RSI = 100 - [100 / (1 + RS)]
其中 RS = 过去N天平均上涨幅度 / 过去N天平均下跌幅度

举个例子（RSI_6，看过去6天）：
  如果6天里涨的多、跌的少 → RS > 1 → RSI > 50
  如果6天里跌的多、涨的少 → RS < 1 → RSI < 50
  如果全是涨、没有跌 → RS → ∞ → RSI → 100（超买）
  如果全是跌、没有涨 → RS = 0 → RSI = 0（超卖）
```

#### 📊 福耀玻璃真实数据（2026-06-18）

| 指标 | 数值 | 含义 |
|---|---|---|
| RSI_2（2日） | **2.43** | 极度超卖（2日周期非常敏感） |
| RSI_6（6日） | **13.70** | 极度超卖（< 30 = 超卖，< 20 = 极度超卖） |
| RSI_12（12日） | **23.58** | 超卖 |
| RSI_24（24日） | **31.63** | 接近超卖阈值（30） |

**判断**：**所有周期的RSI都在超卖区，RSI_6只有13.7，属于极度超卖**。这是判断"超卖"的核心依据。

#### 💡 怎么看RSI——四个核心信号

**信号1：超买 vs 超卖（最基础用法）**

```
RSI > 70  → 超买区（涨太多了，可能回调）
RSI < 30  → 超卖区（跌太多了，可能反弹）

更严格的版本：
RSI > 80  → 极度超买
RSI < 20  → 极度超卖

福耀玻璃：RSI_6 = 13.7 < 20 → 极度超卖
```

**信号2：RSI背离（最有价值的信号）**

```
底背离：股价创出新低，但RSI没有创出新低（在走高）
→ 下跌动能衰竭，即将反弹（这是高胜率信号）

顶背离：股价创出新高，但RSI没有创出新高（在走低）
→ 上涨动能衰竭，即将回调

这是RSI最值钱的信号，比单纯的超买超卖更可靠！
```

**信号3：RSI与中轴（50）的关系**

```
RSI > 50 → 多方力量占优
RSI < 50 → 空方力量占优

福耀玻璃：所有周期RSI都远小于50 → 空方力量绝对占优
```

**信号4：不同周期RSI的配合使用**

```
RSI_6  ：超短期，对价格变化非常敏感，适合短线
RSI_12 ：短期，最常用
RSI_24 ：中期，更平滑，假信号更少

配合用法：
  RSI_6 < 20 且 RSI_12 < 30 → 多重确认超卖，信号更可靠
  
福耀玻璃：RSI_6=13.7，RSI_12=23.58 → 多重确认超卖 ✅
```

#### 💰 与资金流向 + 市场情绪的关系

```
RSI超卖（< 30）：
  → 说明近期卖盘力量远大于买盘
  → 市场情绪：恐慌性抛售
  → 资金流向：短期资金净流出
  → 但！也可能意味着"该卖的都卖了"，抛压即将衰竭

RSI超买（> 70）：
  → 说明近期买盘力量远大于卖盘
  → 市场情绪：贪婪、FOMO（害怕错过）
  → 资金流向：短期资金净流入
  → 但！也可能意味着"该买的都买了"，买盘即将衰竭
```

#### ⚠️ 一个重要警告：超卖可以继续超卖！

很多初学者看到RSI<30就冲进去抄底，结果被套牢。原因是：

**在强下跌趋势中，RSI可以在超卖区"钝化"（长期停留在30以下）**。

```
错误用法：RSI<30 → 立刻买入 ❌

### 2.2 KDJ（随机指标）

#### 📖 来龙去脉

**起源**：KDJ的前身是"随机指标"（Stochastic Oscillator），由George Lane在1950年代发明。后来传入中国，中国的分析师在K和D的基础上，加入了J线，形成了KDJ。

**发明逻辑**：Lane观察到，**在上涨趋势的末期，股价往往还能创新高，但"收盘价相对于最高价的位置"会越来越低**——这说明上涨力气不够了。KD指标就是用来捕捉这个现象的。

**K、D、J分别代表什么**：
- K线（快线）：当前收盘价在过去N天价格波动范围内的相对位置
- D线（慢线）：K线的移动平均（平滑版K线）
- J线：J = 3×K - 2×D（放大版，更敏感）

#### 🔢 怎么算的（直观解释，不写公式）

```
KDJ回答的问题是：
  "今天的收盘价，是落在最近9天价格波动范围的哪个位置？"

如果收盘价接近9天内的最高价 → K值高（> 80）→ 超买
如果收盘价接近9天内的最低价 → K值低（< 20）→ 超卖

J线 = 3K - 2D，是对K和D的"放大"，让信号更敏感
J > 100 → 超买
J < 0   → 超卖（比K和D更极端）
```

#### 📊 福耀玻璃真实数据（2026-06-18）

| 指标 | 数值 | 含义 |
|---|---|---|
| KDJ_K | **18.66** | 快线，< 20，超卖 |
| KDJ_D | **27.82** | 慢线，接近超卖阈值（30） |
| KDJ_J | **0.35** | 极度超卖（< 0 = 极度超卖） |

**判断**：**K、D、J三线都在超卖区，J值只有0.35，极度超卖**。和RSI的信号高度一致！

#### 💡 怎么看KDJ——四个核心信号

**信号1：超买 vs 超卖**

```
K/D > 80  → 超买
K/D < 20  → 超卖
J > 100   → 超买
J < 0     → 超卖

福耀玻璃：K=18.66，D=27.82，J=0.35 → 三线超卖
```

**信号2：金叉 vs 死叉**

```
KDJ金叉：K线从下往上穿越D线 → 买入信号
KDJ死叉：K线从上往下穿越D线 → 卖出信号

注意：在超卖区的金叉，比在中间的金叉更可靠
     在超买区的死叉，比在中间的死叉更可靠

福耀玻璃真实案例：
  2026-05-29出现过一次金叉（K从下往上穿越D）
  当时股价从51.99反弹到54.24，但反弹只持续了3天就失败了
  → 这说明：在下跌趋势中，KDJ金叉的反弹力度和持续性都有限
```

**信号3：J值的极端值**

```
J > 100  → 极度超买，短期回调概率大
J < 0    → 极度超卖，短期反弹概率大
J = 0左右 → 往往是阶段性底部区域

福耀玻璃：J=0.35，接近0 → 极度超卖，反弹概率增大
```

**信号4：KDJ背离（同样是非常有价值的信号）**

```
底背离：股价创新低，但KDJ没有创新低 → 即将反弹
顶背离：股价创新高，但KDJ没有创新高 → 即将回调
```

### 2.3 CCI（商品通道指标 —— Commodity Channel Index）

#### 📖 来龙去脉

**发明者**：Donald Lambert，1980年。
**发明逻辑**：Lambert想衡量"当前价格和它的统计平均价格之间，偏离了多少个标准差"。如果偏离太大，说明价格"不正常"，可能要回归。

CCI最早用于商品期货，但现在股票也常用。

#### 📊 福耀玻璃真实数据（2026-06-18）

| 指标 | 数值 | 含义 |
|---|---|---|
| CCI_14 | **-175.36** | 远低于-100，极度超卖 |

**CCI的判断标准（和RSI不同）**：

```
CCI > +100  → 超买（进入"异常强势"区）
CCI < -100  → 超卖（进入"异常弱势"区）
CCI在-100 ~ +100之间 → 正常波动区

福耀玻璃：CCI = -175.36 < -100 → 极度超卖
```

#### 💡 CCI的特色信号

CCI有一个其他指标没有的信号：**"CCI回归零轴"**。

```
CCI从下方穿越-100向上 → 脱离超卖区，买入信号
CCI从上方穿越+100向下 → 脱离超买区，卖出信号

福耀玻璃：等待CCI从-175.36回升并穿越-100向上 → 确认反弹开始
```

### 2.4 WR（威廉指标 —— Williams %R）

#### 📖 来龙去脉

**发明者**：Larry Williams，1973年。
**发明逻辑**：和KDJ非常类似，也是衡量"收盘价在最近N天价格波动范围内的位置"。WR和KDJ的区别是：**WR用0~-100刻度，KDJ用0~100刻度，方向相反**。

```
WR = -100 × (最高价 - 收盘价) / (最高价 - 最低价)

WR = -20  → 收盘价接近最近N天的最高价（超买）
WR = -80  → 收盘价接近最近N天的最低价（超卖）
```

#### 📊 福耀玻璃真实数据（2026-06-18）

| 指标 | 数值 | 含义 |
|---|---|---|
| WR_6 | **-91.12** | < -80，超卖 |
| WR_10 | **-92.86** | < -80，超卖 |

**判断**：WR同样确认超卖！

### 2.5 BIAS（乖离率）

#### 📖 来龙去脉

**"乖离"**就是"偏离"。乖离率衡量的是：**股价偏离它的均线有多远**？

逻辑很简单：股价不会永远涨，也不会永远跌。如果股价离均线太远，有一种"引力"会把它拉回去。

#### 📊 福耀玻璃真实数据（2026-06-18）

| 指标 | 数值 | 含义 |
|---|---|---|
| BIAS_6 | **-2.84%** | 股价比6日均线低2.84% |
| BIAS_12 | **-4.71%** | 股价比12日均线低4.71% |
| BIAS_24 | **-5.97%** | 股价比24日均线低5.97% |

**判断**：股价向下偏离均线约3~6%，有一定反弹引力，但不是极端（极端一般是BIAS < -8%）。

## 模块三：成交量与资金流向指标
> **核心问题**：价格上涨/下跌，有没有真金白银在支撑？是"虚涨"还是"真涨"？

**技术分析里有一句经典的话："价涨量增，价跌量缩，趋势健康；价涨量缩，价跌量增，趋势可疑。"**

---

### 3.1 成交量 + 换手率（最基础，也最重要）

#### 📖 来龙去脉

**成交量**就是当天有多少股被买卖了。**换手率** = 成交量 ÷ 流通股本，消除了不同股票流通盘大小的影响。

**为什么成交量重要？** 因为价格是"纸面财富"，成交量是"真金白银"。没有成交量配合的价格上涨，往往是"虚涨"（主力自买自卖拉抬，没有真实买盘）。

#### 📊 福耀玻璃真实数据

| 日期 | 收盘价 | 涨跌幅 | 成交量（手） | 成交额（亿元） | 换手率 |
|---|---|---|---|---|---|
| 2026-06-18 | 49.95 | -1.46% | 97,751 | 4.91 | 0.49% |
| 2026-06-17 | 50.69 | +0.08% | 107,461 | 5.40 | 0.54% |
| 2026-06-16 | 50.65 | -3.35% | 163,843 | 8.38 | 0.82% |
| 2026-06-15 | 52.41 | +0.11% | 103,292 | 5.40 | 0.52% |
| 2026-06-12 | 52.35 | -0.11% | 102,578 | 5.38 | 0.51% |

**判断**：
- 6月16日大跌3.35%，成交量放大到16.38万手（换手率0.82%）→ **放量下跌**，卖盘集中涌出，偏空信号
- 近5日换手率0.4~0.8%，整体偏低 → 市场关注度不高，交易不活跃
- 没有出现"放量上涨"的信号 → 反弹还没有资金确认

#### 💡 成交量的核心信号

```
健康上涨：价涨 + 量增 → 有资金在追涨，趋势可持续
可疑上涨：价涨 + 量缩 → 没有新增资金，可能是反弹而非反转

健康下跌：价跌 + 量缩 → 卖盘逐渐枯竭，可能即将见底
危险下跌：价跌 + 量增 → 资金在集中出逃，跌势可能继续

福耀玻璃：6月16日"价跌+量增" → 危险下跌，资金在出逃
```

### 3.3 融资余额（A股特色指标）

#### 📖 来龙去脉

A股有"融资融券"制度——投资者可以向券商借钱买股票（融资），或者借股票卖出（融券）。
**融资余额** = 所有投资者借钱买入这只股票的总金额。

**融资余额的意义**：
- 融资余额上升 → 杠杆资金在进场，市场情绪偏乐观（但也可能意味着"散户在高位接盘"）
- 融资余额下降 → 杠杆资金在离场，或被迫平仓（踩踏风险）

### 3.4 股东户数（筹码集中度）

#### 📖 来龙去脉

**股东户数**告诉你这只股票有多少个人/机构持有。**户数的变化**告诉你筹码是在集中还是分散。

```
户数增加 → 筹码从机构/大户分散到散户 → 通常偏空（散户接盘）
户数减少 → 筹码从散户集中到机构/大户 → 通常偏多（聪明钱在收集）
```

#### 📊 福耀玻璃真实数据

| 户均持股 | **约1.5万股** | 下降 |

**判断**：股东户数大幅增加，说明筹码在分散，散户在接盘，偏空信号。

## 附录：初学者十大常见误区

> **为什么要有这个附录？** 技术分析是一门"看起来简单，做起来难"的技能。初学者往往会犯一些典型的错误，导致亏损。我把这些错误总结成"十大误区"，希望你能够避免。

---

### 误区1：把技术指标当"水晶球"

```
❌ 错误做法：
   看到RSI<30，立刻全仓买入，期待明天就反弹
   
✅ 正确做法：
   RSI<30只是说"下跌动能可能衰竭"
   反弹什么时候来、幅度多大，技术指标回答不了
   需要结合：趋势、成交量、资金流向综合判断
```

**福耀玻璃真实案例**：
- 2026年6月5日：RSI_6=28.5（超卖），有初学者可能买入
- 结果：股价从53.60继续跌到49.95（-6.8%）
- 教训：RSI超卖不等于立刻反弹，超卖可以继续超卖！

---

### 误区2：只看一个指标，不看其他指标

```
❌ 错误做法：
   只看RSI，发现超卖就买入，不看均线、MACD、成交量
   
✅ 正确做法：
   多个指标互相验证
   如果 RSI超卖 + MACD底背离 + 成交量放大 + 均线金叉
   四个信号一起出现，可靠性才高
```

**福耀玻璃真实案例**：
- 2026年6月18日：RSI超卖 ✅，但均线死叉 ❌，MACD死叉 ❌，成交量未放大 ❌
- 结论：只有一个指标发出买入信号，其他都偏空 → 不要买！

---

### 误区3：在下跌趋势中盲目抄底

```
❌ 错误做法：
   "股价从60跌到50了，跌了17%，肯定到底了，买入！"
   
✅ 正确做法：
   股市名言："趋势是你的朋友"（Trend is your friend）
   下跌趋势中的超卖反弹，往往力度弱、时间短
   真正的底部，需要趋势指标（均线、MACD）也给出转向信号
```

**福耀玻璃真实案例**：
- 2026年2-6月：股价从63元跌到49.95元（-21%）
- 很多初学者在57元、55元、53元抄底，结果都被套牢
- 教训：不要猜底，等待趋势反转信号再买入

---

### 误区4：看到金叉就买，看到死叉就卖

```
❌ 错误做法：
   看到MA5上穿MA20形成金叉 → 立刻买入
   看到MA5下穿MA20形成死叉 → 立刻卖出
   
✅ 正确做法：
   先判断当前是上涨趋势还是下跌趋势
   - 上涨趋势中：金叉可靠，死叉可能是回调
   - 下跌趋势中：死叉可靠，金叉可能是反弹（陷阱）
```

**福耀玻璃真实案例**：
- 过去4个月：3次金叉，2次失败（成功率33%）
- 过去4个月：3次死叉，3次成功（成功率100%）
- 教训：在下跌趋势中，死叉比金叉可靠得多！

---

### 误区5：忽视成交量，只看价格

```
❌ 错误做法：
   看到股价上涨就兴奋，不看成交量是否配合
   
✅ 正确做法：
   价涨量增 = 健康上涨（有资金推动）
   价涨量缩 = 可疑上涨（可能是反弹，不是反转）
   价跌量增 = 危险下跌（资金在出逃）
   价跌量缩 = 可能见底（卖盘枯竭）
```

**福耀玻璃真实案例**：
- 2026年6月16日：大跌3.35%，成交量放大到16.38万手
- 解读：价跌+量增 = 资金在集中出逃，偏空信号
- 结果：接下来两天继续下跌

---

### 误区6：频繁交易，过度操作

```
❌ 错误做法：
   每天盯着盘面，看到小波动就买卖
   今天买这个，明天卖那个，频繁换股
   
✅ 正确做法：
   技术指标的 signals 不是天天有
   有价值的信号：一周可能只有1-2次
   大部分时间应该：观察、等待、学习
```

**福耀玻璃实战建议**：
- RSI超卖信号：平均1-2个月出现一次
- 均线金叉信号：平均1-2个月出现一次
- 不要每天都交易，耐心等待高胜率信号

---

### 误区7：不设止损，死扛亏损

```
❌ 错误做法：
   买入后股价下跌，舍不得止损，想着"总会涨回来的"
   结果越套越深，从套牢5%到套牢20%
   
✅ 正确做法：
   买入前先设定止损位（比如：-5%止损）
   一旦触发止损，坚决执行，不要犹豫
   "截断亏损，让利润奔跑"
```

**福耀玻璃真实案例**：
- 假设你在2026年4月9日金叉时买入（56.48元）
- 如果没有止损，到6月18日已经亏损11.6%
- 如果有止损（-5%），可以在53.66元止损，只亏5%

---

### 误区8：追涨杀跌，情绪化交易

```
❌ 错误做法：
   看到股价大涨，FOMO（害怕错过），追进去
   看到股价大跌，恐慌，割肉在底部
   
✅ 正确做法：
   制定交易计划，按计划执行
   大涨后（RSI>70）：应该考虑卖出，不是买入
   大跌后（RSI<30）：应该观察等待，不是恐慌卖出
```

**福耀玻璃真实案例**：
- 2026年1月7日：股价涨到63.33元（阶段性高点）
- 如果当时FOMO追进去，到现在已经亏损21%
- 正确做法：RSI超买时应该卖出，不是买入！

---

### 误区9：盲目相信"专家"、"大V"的推荐

```
❌ 错误做法：
   看到某大V推荐某只股票，立刻买入
   不看基本面，不看技术面，盲目跟风
   
✅ 正确做法：
   别人的推荐可以参考，但要自己验证
   问自己：当前趋势是什么？技术指标怎么说？资金流向如何？
   只有自己理解的逻辑，才能拿得住
```

**福耀玻璃实战建议**：
- 不要因为某大V说"福耀玻璃是价值投资标的"就买入
- 要自己看懂：当前技术面偏空，但基本面（PE=14x）有吸引力
- 制定自己的策略：价值投资可以分批建仓，但不要一次全仓

---

### 误区10：不记录、不总结、不学习

```
❌ 错误做法：
   买卖全凭感觉，不记录交易理由
   亏了也不知道为什么亏，赚了也不知道为什么赚
   重复犯同样的错误
   
✅ 正确做法：
   记录每一笔交易：买入理由、卖出理由、结果
   定期复盘：哪些做对了？哪些做错了？
   不断学习：看书、看教程、实践、总结
```

**福耀玻璃实战练习**：
- 记录：2026-06-18，RSI超卖，买入，价格49.95
- 跟进：5天后（2026-06-25）查看结果，记录涨跌
- 总结：这个信号可靠吗？下次还要不要信？
